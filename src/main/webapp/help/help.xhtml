Two types of webhooks exist in eventlogger.

http - for plain text messages

fluentd-syslog - for syslog messages from fluentd in json or ndjson format

http - for plain text messages 

GET  /eventlogger/api/v1/http?tag=json&ident=my-process&msg=This is the text message

or

 
POST  /eventlogger/api/v1/http?tag=json&ident=my-process
Body: {"anything":1,"or":"just plain text"}
Generic HTTP endpoint accepts any GET or POST http request. Message is taken from URL params in case of GET request or from the body of POST request. For GET requests the URL params must be set as:
